# CISA KEV Remediation Guidance

Known exploited vulnerabilities should be treated as urgent because there is evidence of active exploitation.

Recommended actions:
- Prioritise remediation immediately.
- Confirm whether affected products are present in the environment.
- Apply vendor patches or mitigations as soon as possible.
- If patching is not immediately possible, isolate affected systems.
- Increase monitoring for related indicators of compromise.
- Review access logs, authentication logs, and network activity.
- Validate that remediation has been completed.
- Document the remediation owner, deadline, and verification evidence.

Operational priority:
- Known exploited + internet-facing system = emergency response.
- Known exploited + critical CVSS score = emergency response.
- Known exploited + network attack vector = urgent remediation.
